# Warframe Progress Tracker

A comprehensive tool to track your progress across all Warframe items including:
- Primary/Secondary/Melee weapons
- Warframes
- Kitguns & Zaws
- Companions
- Necramechs (Voidrigs)
- Archwings & Archguns
- Mods & Robotics
- Archmelees & Amps
- Incarnon weapons
- Vehicles

## Project Structure
- `scraper/` - Web scraping scripts for Warframe wiki data
- `data/` - Raw and processed data storage
- `database/` - Database schema and setup
- `tracker/` - Progress tracking application
- `config/` - Configuration files
